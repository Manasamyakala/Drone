# Counter System for AFVs

## Overview
The AI-Powered Counter System for Military Vehicles (AFVs) is an intelligent defense solution designed to protect armored vehicles such as tanks and combat vehicles from hostile drones. The system combines Artificial Intelligence, computer vision, and sensor fusion to detect, track, classify, and assess aerial threats in real time.

By continuously monitoring the surrounding environment through cameras and sensors, the system identifies different types of drones—including surveillance, attack, and swarm drones—and evaluates their threat level. Based on the analysis, it recommends or simulates appropriate countermeasures such as signal jamming (soft kill), interception, or instant threat alerts.

The primary objective of this project is to enhance battlefield survivability by providing rapid, automated, and intelligent drone threat detection and response.

---

## Key Features

-  Drone classification (Surveillance, Attack, Swarm)
-  AI-based threat assessment and prioritization
-  Real-time target tracking
-  Countermeasure recommendation
-  Signal Jamming (Soft Kill)
-  Interception Simulation
-  Threat Alerts
-  Automated decision support system

---
## System Workflow

1. Capture live video and sensor data.
2. Detect drones using AI-based object detection models.
3. Track detected drones in real time.
4. Classify drone type and estimate threat level.
5. Recommend or simulate the most suitable countermeasure.
6. Notify operators through a monitoring dashboard.
